# Mouton

## What is it ?

Mouton is a 3D engine.

## How to install ?

You will need a C++17 compiler.

Clone this repository

```
git clone https://gitlab.com/Volham/mouton.git
```
Then use cmake to create makefiles
```
cd mouton && cmake .
```
Compile the whole source code
```
make
```
